# FTIR スペクトル画像抽出ツール

**画像から赤外スペクトル曲線を抽出し、波数-強度 CSV データに変換します。**

**FTIR（フーリエ変換赤外分光法、Fourier Transform Infrared Spectroscopy）** スペクトルのデジタル化に特化——論文・スクリーンショット・スキャン図版から波数（cm⁻¹）対 吸光度/透過率データを抽出します。完全なインタラクティブ Web インターフェースを提供：画像アップロード、スペクトル領域のクロップ、IR 軸パラメータ設定、クリーンな CSV エクスポート。

> **他の分光法・クロマトグラフィー技術への適用：**
> 本プロジェクトは赤外スペクトル（IR / FT-IR / Mid-IR / NIR / Far-IR / ATR-FTIR）専用に開発されています。**ラマン分光法**、**UV-Vis（紫外可視分光法）**、**GC（ガスクロマトグラフィー）**、**HPLC**、**GC-MS**、**LC-MS**、**XRD（X線回折）**、**NMR（核磁気共鳴）**、**蛍光分光法**、**TGA（熱重量分析）**、**DSC（示差走査熱量測定）** などの分析曲線を扱う場合は、本プロジェクトを fork して軸マッピング（X軸：波長、保持時間、2θ、化学シフト、温度、m/z；Y軸：強度、応答、重量減少%）を変更してください。コアの画像抽出アルゴリズム（適応的閾値、ベイズ色分類、3次補間）は分析手法に依存しません。

## 機能一覧

| 方法 | 仕組み |
|------|--------|
| **自動抽出** | 適応的閾値 + 輪郭解析。12 モードが二値化方向と閾値レベルを切り替え — 複数回クリックして最良の結果を選択。 |
| **カラー抽出** | ベイズ色分類。曲線上のシード領域を選択；アルゴリズムが色分布から曲線ピクセルと背景を分離。 |
| **手動トレース** | 曲線に沿ってガイドポイントを配置；3次スプライン補間で全ガイドを厳密に通る約 200 点を生成。 |

pip install -r requirements.txt

```

```

## ワークフロー（インタラクティブ Web UI）

1. **アップロード** — スペクトル画像をアップロード（JPG/PNG、900px に自動リサイズ）
2. **クロップとパラメータ** — クロップ枠をスペクトル領域に合わせる；波数範囲、データタイプ、オプションの 2000 cm⁻¹ 軸分割を設定
3. **背景除去** — 背景領域を選択してグリッド線とノイズを抑制
4. **抽出** — 自動 / カラー / 手動トレースを選択；曲線をプレビュー；消しゴムと波数オフセットで調整；CSV ダウンロード

## 対応範囲と関連技術

本ツールは赤外分光法の曲線抽出を**直接サポート**します：

- FTIR、FT-IR、ATR-FTIR、DRIFTS、透過 IR、反射 IR
- 中赤外（4000–400 cm⁻¹）、近赤外（NIR）、遠赤外（Far-IR）
- 吸光度、透過率、Kubelka-Munk
- 波数（cm⁻¹）軸、2000 cm⁻¹ での非線形分割対応

**fork して軸パラメータを変更することで適用可能**：

- ラマン分光法（ラマンシフト cm⁻¹、SERS、共鳴ラマン）
- UV-Vis 分光法（波長 nm、光学密度）
- クロマトグラフィー（GC、HPLC、UHPLC、GC-MS、LC-MS、保持時間）
- 熱分析（TGA、DSC、DTA、温度 °C、重量減少）
- XRD（2θ）、XRF、SAXS
- NMR（¹H、¹³C、化学シフト ppm）
- 質量分析（m/z）
- 蛍光、フォトルミネッセンス、発光分光法
- 電気化学（サイクリックボルタンメトリー、インピーダンス分光法）

## FTIR.fun について

このツールは **[FTIR.fun](https://ftir.fun)** のオープンソース抽出エンジンです — 13 万件以上の FTIR 参照スペクトルを持つクラウド赤外分光分析プラットフォーム。

[ftir.fun](https://ftir.fun) でできること：
- 抽出した CSV をアップロードし、スペクトルライブラリで材料を検索
- 化学ナレッジグラフに基づく AI ピーク解説を取得
- 共有可能なトライアクシスレポートを生成
- 28 以上の測定ファイル形式を解析

**オンラインで試す：** https://ftir.fun/spectrum/upload-image/
